Features

Installation

Choose your setup path

Recommended

Download ZIP

  1. Download the extension ZIP file.
  2. Extract it to a folder.
  3. Open chrome://extensions.
  4. Enable Developer mode.
  5. Click Load unpacked and select extracted folder.
Optional

Chrome Web Store

If a public store URL is configured, install in one click.

FAQ

Common questions

Does it collect browsing data?

No, core protection and stats run locally in your browser.

Will it slow down Chrome?

It is built on low-overhead Manifest V3 APIs.

Can I disable it per site?

Yes, per-domain controls are available in popup and dashboard.

Is it free?

Yes, the current project is free and open-source.
