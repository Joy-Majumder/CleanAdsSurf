Frequently Asked Questions

Is CleanAdsSurf safe?

Yes! We use Chrome's declarativeNetRequest API which is sandboxed and secure. We never collect your data.

Will it slow down my browser?

No, it's optimized for performance. The extension uses minimal resources and barely impacts browsing speed.

Does it work on all websites?

Yes, it works on all websites. You can disable it per-domain if a site has issues.

Is my data safe?

100% safe. All processing happens locally on your device. Nothing is sent to external servers.

Can I use it on mobile?

Currently available on desktop Chrome only. Mobile support is planned for future versions.

Is there a premium version?

No, CleanAdsSurf is completely free with all features included.
